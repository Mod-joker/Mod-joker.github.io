由万事开头难想到的

发表于 2017-02-14 | 分类于 个人感悟 | | 阅读次数
万事开头难，但是事情开了头并不等于成功的一半，还需要坚持到底的毅力加上行动才能达到当初所立的目标。

工作两年，收获多少有些，加上欠自己的总结，产生了这篇文字。之前读书时有写东西记录习惯，工作后偶尔冒出来记录的年头，自己总是以或这或那的理由让自己把这年头掐灭在萌芽状态，念头最盛的时候也曾注册域名并绑定github pages创建了博客，但最终以未写一篇而不了了之，一年后域名到期且也换了新主机，还是未记一字。遂悟人之惰性实是与生俱来，如不克服，真或难成一事。又从头申请域名来做这件事情，并警示自己要坚持到底。伟人曾言：苟有恒，何必三更起五更眠;最无益，只怕一日曝十日寒。谨以此自警、自勉、自励。

工作收获

  1. 很多事情，不能急于着手去完成，不妨科学的分步骤解决；个人建议如下，梳理事情的脉络，找出解决问题的关键，把关键难题逐渐分解，逐步解决问题如遇困难看计划安排来决定是否先自身尝试解决或求助。古人有训如下

    撼大摧坚，要徐徐下手，久久见功，默默留意，攘臂极力，一犯手自家先败。﻿ -明.吕坤 《呻吟语》

  2. 兴趣的重要性。做一件事，能长期坚持下来的最主要的动力就是兴趣，找到感兴趣的事，坚持到底。

    人若志趣不远，心不在焉，虽学无成。

  3. 关于选择。一个的人选择在很大程度上决定他的高度，但是选择的诸多因素源于自身对外界的各种判断，充实自己，不断汲取各种知识，在选择的时候往往能给你正确的方向。在犹豫不定时，如果时间充足，谋定而后动，多方面比较，不必过快决定；若时不待我，则不妨选择挑战性更强的，认真对待，失败也无悔。对工作的选择尤其如是。

    我来不及认真地年轻，待明白过来时，只能选择认真地老去。

写在2017的开头

  1. 毕业了两年半，欠了两个年终给自己的交代，经济方面没有大的起色，工作也是不温不热，缺乏很大的热情，这些都需要自己慢慢纠正。年初给自己定一个小目标，由于最近对人工智能方面比较感兴趣，故决定，今年系统的学习一下Deep Learning相关的知识，选择Python作为实践语言，在网上学习，最终能做一个东西应用到实际生活中来。并且工作之余要积极健身锻炼，保持身体的活力和健康，把丢掉的爱好捡起来，坚持练字。
  2. 时常浏览网上的科技消息，真真觉得，学如逆水行舟，不进则退。如今的技术发展之快，人们要保持跟上科技的脚步需要付出很大的精力，所以日常的琐碎时间要高效的利用来学习一些新的有趣的知识。
  3. 坚持阅读。鸟欲高飞先振翅，人求上进先读书。阅读是充实自己的最有效手段，不可废弃。

最后

第一篇博文，希望再接再厉。2017年努力前行。
